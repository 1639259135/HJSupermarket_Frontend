- **详情**:

- **验收**: `npm run dev` 启动成功，空白页正常渲染

### F-002 — Axios 封装与统一错误处理

- **状态**: pending
- **优先级**: P0
- **依赖**: F-001
- **详情**:
  - 创建 `src/api/request.ts`：baseURL + 超时(30s) + 请求拦截器（注入 Token） + 响应拦截器
  - 响应拦截器统一处理：`code:200` → 返回 `data`；`code:401` → 清除 Token → 跳转登录页；其他 → `message.error()` 提示
  - 封装 `get/post/put/delete` 泛型方法
  - 错误码映射函数：`{400:"请检查输入", 401:"登录已过期", 403:"权限不足", 2002:"库存不足", ...}`
- **验收**: 发起一个 401 请求 → 自动跳转 `/login`；发起一个 400 请求 → Toast 提示中文错误

### F-003 — 布局框架与路由

- **验收**: 登录后左侧菜单显示对应权限的菜单项；点击菜单跳转正确页面

### F-004 — 登录页 `/login`
